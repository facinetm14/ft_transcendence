# TECH STACK
<ul>
  <li><strong>Programming Language: </strong> TypeScript</li>
  <li><strong>Front: </strong>React</li>
  <li><strong>Back: </strong>NestJS</li>

</ul>

# STOP
`docker-compose down`

Open Browser of your choice (built with Chromium in mind) and go to 'http://localhost:3000/'
